# 主界面卡牌操控规则

**模块名称：** 主界面卡牌操控规则  
**功能：** 定义主界面元素和对应每种卡牌在主界面元素上的操控规则

---

## 一、主界面卡牌种类

### 1. 单词卡

| 界面元素 | 显示位置 |
|---------|---------|
| 单词卡信息 | 卡牌中间 |

### 2. 分类卡

| 界面元素 | 显示位置与规则 |
|---------|--------------|
| 分类卡标识 | 卡牌左上角 |
| 分类卡对应词组中单词卡数量 | 卡牌右上角，格式为 `0/N`；每放置 1 张单词卡后变更为 `1/N`，直至 `N/N` 后播放卡包收起动画并移除分类卡 |
| 分类卡文本信息 | 卡牌中间 |

### 3. 万能卡

- 显示万能卡图片，图片上**无文本信息**

---

## 二、主界面区域

### 1. 列区

#### 说明
- 主界面有 N 列列区，列的数量由**关卡配置表中棋牌列数**决定

#### 界面规则
- 关卡中每个列区可配置默认卡牌数量
- 卡牌从上到下排列，**y轴位置最下方**的卡牌为**翻开状态**，可见全部信息且可拖拽
- 其余卡牌均为**卡背状态**，不可查看信息，不可拖拽
- 翻开的卡牌被成功拖拽放置后，下一张卡背状态的卡牌变为翻开状态；若已无卡牌，则该列区变为**空白状态**

#### 列区状态

| 状态 | 说明 |
|------|------|
| 列区-空白 | 列区没有任何卡牌 |
| 列区-分类卡 | 列区最下面一张为分类卡，只能移动到空白的分类槽或万能卡上 |
| 列区-单词卡 | 列区最下面一张为单词卡，可拖拽放置到同词组的单词卡上或分类槽上的分类卡上，不能拖拽到列区-分类卡上 |
| 列区-万能卡 | 列区最下面一张为万能卡，允许拖拽到其他列区位置 |

#### 操控规则

**列区-分类卡**

- 点击拖拽时，选中该分类卡及同词组的所有卡牌一同拖拽
  - 若一同拖拽的卡牌数量 > 1：卡牌折叠在一起，只显示最下方单词卡的文本信息，右上角显示本次拖拽的卡牌数量
  - 若卡牌数量仅为 1：不显示数字

| 目标位置 | 操作结果 | 判定 |
|---------|---------|------|
| 空白的分类槽 / 空白的列区 / 万能卡上 | ✅ 允许放置，步数 -1 | 有效操作 |
| 列区-单词卡（同词组） | ✅ 允许放置，步数 -1 | 错误操作 |
| 列区-单词卡（不同词组） | ❌ 无法移动，提示：**分类卡只能放在与其类别相符的词语上** | 错误操作 |
| 列区-分类卡 | ❌ 无法移动，提示：**此列以分类卡开始，你只能将卡槽上将卡牌堆叠在分类卡上** | 错误操作 |
| 分类槽（已有卡牌放置） | ❌ 拖拽无效，自动还原到原位置 | 错误操作 |
| 其他区域 | ❌ 拖拽无效，自动还原到原位置 | 无效操作 |

---

**列区-单词卡**

- 点击拖拽时，选中列区最下方的单词卡，并将该列中同词组的所有卡牌一同选中
  - 若选中卡牌数量 > 1：卡牌折叠，只显示最下方单词卡的文本信息，右上角显示本次拖拽的卡牌数量
  - 若卡牌数量仅为 1：不显示数字

| 目标位置 | 操作结果 | 判定 |
|---------|---------|------|
| 分类槽（已放置相同词组的分类卡或单词卡） | ✅ 允许放置，步数 -1 | 有效操作 |
| 分类槽（空白） | ❌ 无法放置，提示：**只有分类卡才能放置在分类槽上** | 错误操作 |
| 分类槽（已放置不同词组的卡牌） | ❌ 无法放置，自动还原到原位置 | 错误操作 |
| 列区-空白 | ✅ 允许放置，步数 -1 | 有效操作 |
| 列区-分类卡（同词组） | ✅ 允许放置，步数 -1 | 有效操作 |
| 列区-分类卡（不同词组） | ❌ 无法移动，提示：**此列以分类卡开始，你只能将卡槽上将卡牌堆叠在分类卡上** | 错误操作 |
| 列区-万能卡 | ✅ 允许放置，步数 -1 | 有效操作 |
| 列区-单词卡（同词组） | ✅ 允许放置，步数 -1 | 有效操作 |
| 列区-单词卡（不同词组） | ❌ 无法移动，提示：**你只能堆叠同一词组的卡牌** | 错误操作 |
| 其他区域 | ❌ 拖拽无效，自动还原到原位置 | 无效操作 |

---

**列区-万能卡**

- 选中万能卡拖拽时：
  - 若最下方**只有万能卡**：可移动到其他列区
  - 若最下方**还有其他分类卡或单词卡**：❌ 无法移动，提示：**无法使用万能卡来移动牌堆**

| 目标位置 | 操作结果 | 判定 |
|---------|---------|------|
| 列区-空白 | ✅ 允许放置，步数 -1 | 有效操作 |
| 列区-分类卡 | ✅ 允许放置，步数 -1 | 有效操作 |
| 列区-单词卡 | ✅ 允许放置，步数 -1 | 有效操作 |
| 分类槽 | ❌ 无法放置，提示：**万能卡不能放置在分类槽上** | 错误操作 |
| 万能卡上 | ❌ 无法放置，提示：**万能卡不能叠加** | 错误操作 |
| 其他区域 | ❌ 拖拽无效，自动还原到原位置 | 无效操作 |

---

### 2. 分类槽

#### 说明
- 主界面有 N 个分类槽，数量由**关卡配置表中的卡槽数量**决定

#### 界面规则

| 状态 | 显示内容 |
|------|---------|
| 空白 | 显示默认分类槽图片 |
| 有分类卡，无单词卡 | 显示分类卡完整信息：左上角为分类卡标识，右上角为 `0/N`（N 为该词组最大卡牌数量），中间为分类卡文本信息 |
| 有分类卡，有单词卡 | 显示最新放置的单词卡信息；单词卡左上角显示 `Y/N`（Y 为当前已放置数量，N 为该词组最大数量），分类槽上方新增分类卡文本信息 |
| 分类卡对应单词卡数量已满 | 播放卡包收集动画，分类槽变更为空白状态 |

#### 操控规则
- 分类槽为**空白**时：只能放置分类卡，放置后不允许拖拽移动
- 分类槽有分类卡后：只能放置与分类卡**相同词组**的单词卡

---

### 3. 牌库

#### 说明
- 关卡内所有词组卡牌中，未放置在列区和手牌区的**剩余卡牌**
- 牌库中的卡牌在关卡开始时由洗牌算法生成固定顺序序列，之后不再改动
- 牌库默认为**卡背状态**
- 牌库右上角显示当前牌库中的卡牌总数量

#### 操控规则
- 点击牌库卡牌：从牌库顺序抽出 1 张手牌至手牌区，牌库数量 -1，步数 -1
- 当最后一张卡牌被抽出后，牌库显示**"恢复库存"**文本
  - 点击该文本：将手牌区所有手牌按初始顺序放回牌库并恢复为卡背状态，步数 -1

---

### 4. 手牌区

#### 说明
- 从牌库抽取的手牌展示区
- 每获得 1 张手牌，手牌区数量 +1，**最多显示 3 张**，超过 3 张时将之前的卡牌隐藏
- **只有最新的一张手牌允许拖拽**
- 手牌被成功拖拽至分类槽或列区后，手牌区数量 -1，后续手牌自动向前补位；若有隐藏手牌，则按顺序从隐藏手牌中抽出显示

#### 界面规则

| 状态 | 显示方式 |
|------|---------|
| 无手牌 | 显示默认空白区域 |
| 1 张手牌 | 完整显示该手牌内容 |
| 2 张手牌 | 最新手牌完整显示，前一张手牌隐藏在后面只显示侧边部分，内容竖向显示在侧边 |
| 3 张及以上手牌 | 只显示 3 张：最新手牌完整显示，前 2 张重叠只显示侧边部分，内容竖向显示在侧边；第 3 张之前的手牌隐藏 |

#### 操控规则
- 只能拖拽**最新的手牌**，成功放置后步数 -1，下一张手牌变为最新手牌可供拖拽
- 拖拽规则与**列区的卡牌操控规则**相同

---

## 三、连击计算

- **连击**：每完成 1 次**有效操作**，连击次数 +1
- 期间不允许有**错误操作**，若发生错误操作，则当前累计的连击次数**重置为 0**
- **无效操作**不影响连击次数计算

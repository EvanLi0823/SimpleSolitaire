## 一、严重漏洞（阻碍游戏正常运行）

### 1.1 分类卡类型缺失

**文档要求** (2.2节):
- 分类卡是独立的卡牌类型，用于标识词组类别
- 可放置在空白分类槽中，激活该槽位
- 可拖拽到列区作为起始卡牌

**当前实现**:
```csharp
// CardType.cs - 缺少CategoryCard类型
public enum CardType
{
    Text,    // 文字卡牌
    Image,   // 图片卡牌
    Joker    // 万能卡
}
```

**影响**: 游戏核心玩法无法实现，玩家无法激活分类槽

**修复建议**:
```csharp
public enum CardType
{
    Text,          // 文字卡牌
    Image,         // 图片卡牌
    Joker,         // 万能卡
    CategoryCard   // 分类卡（新增）
}
```

---

### 1.2 分类卡创建逻辑完全缺失

**文档要求** (2.3.2节):
- 分类卡应作为独立卡牌存在
- 可出现在牌库、列区、手牌区
- 每个词组有且只有一张分类卡

**当前实现** (`WordSolitaireCardLogic.cs:115-142`):
```csharp
private void InitializeCards()
{
    // 只创建了单词卡，完全没有创建分类卡的逻辑
    foreach (WordItem wordItem in _currentLevelWords)
    {
        WordSolitaireCard card = CreateWordCard(wordItem);
        // ...
    }
    // 所有卡牌都放入牌库，没有分类卡
}
```

**影响**: 玩家永远看不到分类卡，无法开始游戏核心流程

**修复建议**:
1. 修改 `WordCategoryData` 添加 `CategoryCard` 数据
2. 在 `InitializeCards()` 中为每个词组创建分类卡
3. 实现分类卡与单词卡的关联逻辑

---

### 1.3 列区初始化逻辑错误

**文档要求** (2.3.2节):
- 列区初始时应有卡牌（配置在 `InitialCardsPerColumn`）
- 最下方卡牌翻开，其余为卡背
- 可能包含分类卡作为起始

**当前实现** (`WordSolitaireCardLogic.cs:115-142`):
- 所有卡牌都放入牌库 (`PackDeck`)
- 列区 (`ColumnDecks`) 初始为空
- 未读取 `LevelData.InitialCardsPerColumn` 配置

**影响**: 游戏开始时列区空白，与产品设计完全不符

**修复建议**:
```csharp
private void InitializeCards()
{
    // 1. 创建所有卡牌（包括分类卡）
    // 2. 根据 InitialCardsPerColumn 分发到列区
    // 3. 剩余卡牌放入牌库
    // 4. 设置列区卡牌状态（最下方翻开，其余卡背）
}
```

### 1.4 分类槽准入逻辑不完整

**文档要求** (2.3.3节):
- 空白分类槽**只能接受分类卡**
- 有分类卡后**只能接受同词组单词卡**
- 分类卡放入后**不允许再拖拽移动**

**当前实现** (`WordSolitaireDeck.cs:44-75`):
```csharp
public override bool AcceptCard(Card card)
{
    // ...
    case WordDeckType.CategorySlot:
        // 分类槽已满，不接受
        if (IsComplete) return false;
        // 分类槽：类别必须匹配，或者是万能卡
        if (wordCard.IsJoker) return true;
        return wordCard.CategoryId == CategoryId;
}
```

**问题**:
1. 空白分类槽接受任何匹配的卡牌（包括单词卡），违反规则
2. 没有检查分类卡放入后不可移动的逻辑
3. 万能卡应该**不能**放入分类槽（文档 2.3.2节）

**修复建议**:
```csharp
case WordDeckType.CategorySlot:
    if (IsComplete) return false;
    
    // 空白槽：只接受分类卡
    if (!HasCards)
        return wordCard.WordCardType == CardType.CategoryCard;
    
    // 有分类卡后：只接受同词组单词卡（拒绝万能卡）
    return wordCard.CategoryId == CategoryId && 
           wordCard.WordCardType == CardType.Text || wordCard.WordCardType == CardType.Image;
```

## 核心玩法

**Word Solitaire: Categories** 是一款将经典**接龙纸牌游戏**与**词汇学习**相结合的益智休闲游戏，口号为 "Play & Relax"（玩中放松）。

- 游戏基于标准接龙（Solitaire）规则，玩家依次翻开牌堆中的卡牌
- 每张卡牌对应一个**主题词汇**（如水果、动物、音乐等类别中的单词）
- 卡牌配有该单词的**英文释义**（Definition），玩家可点击查看含义，兼具词汇学习功能
- 每局有**步数限制**（Moves），耗尽步数则挑战失败

## 难度设置

游戏提供多种难度等级，满足不同玩家需求：

| 难度 | 说明 |
|------|------|
| Simple（简单） | 入门级，步数宽裕 |
| Normal（普通） | 标准难度 |
| Hard（困难） | 步数减少，挑战增加 |
| Super Hard（超级困难） | 高强度挑战 |
| Hardcore（极限） | 最高难度，几乎无容错空间 |

## 道具与助手系统

| 道具 | 功能描述 |
|------|----------|
| **Hint（提示）** | 高亮可操作的卡牌，帮助玩家找到出路，消耗提示次数 |
| **Joker Card（万能牌）** | 可放置在任意卡牌上，也可在其上放置任意卡牌（"The Joker Card can be placed on any card and allows to be placed on it"） |
| **Shuffle（洗牌）** | 重新洗牌并排布可用卡牌，消耗洗牌次数 |
| **Undo（撤销）** | 撤回上一步操作，消耗撤销次数 |
| **Add Moves（增加步数）** | 步数耗尽时，可花费金币或观看激励广告获得额外步数 |

### 付费商品

| 道具礼包（Boosters Offer） | 捆绑销售提示、洗牌等道具组合 |

### 免费获取
- 观看**激励广告**可获得金币、道具或额外步数
- 使用免费 Joker 牌（FreeJoker）

## 界面与体验

### 视觉主题
- **浅色主题**（Light/Default）
- **深色主题**（Dark）
- **跟随系统**（System）

### 操作布局
- **标准布局**：默认双手操作
- **单手布局**（One-handed layout）：适合单手游玩
- **左手布局**（Left-handed layout）：镜像布局适配左利手玩家

### 体验细节
- 卡牌翻牌、洗牌有流畅专属动画（DOTween）
- 支持**振动触觉反馈**（Lofelt NiceVibrations）
- 词汇释义弹窗（Definition Window），随时查看单词含义

## 一、产品概述

### 1.1 产品定位

将经典纸牌接龙玩法与词语联想知识性相结合的休闲益智手游。玩家在限定步数内，识别单词卡牌的主题类别，将其拖放到正确的类别槽中，清空所有牌堆即可过关。

### 1.2 核心价值主张

| 维度 | 描述 |
|------|------|
| **娱乐性** | 经典纸牌拖拽玩法，轻松上手，解压休闲 |
| **知识性** | 词汇分类挑战，拓宽知识面，寓教于乐 |
| **挑战性** | 步数限制制，需要策略规划与词汇积累 |
| **全球性** | 英文词库为基础，多语言本地化支持 |

### 1.3 目标用户

- 休闲益智游戏爱好者
- 英语学习者/词汇积累者
- 全年龄段玩家（6岁+）

### 2.3 基本操作规则

#### 2.3.1 操作消耗规则

| 操作 | 描述 | 消耗 |
|------|------|------|
| **翻牌** | 点击牌库翻一张卡牌到手牌区 | 消耗1步 |
| **拖拽到分类槽** | 从手牌区或列区拖拽卡牌到分类槽 | 消耗1步 |
| **拖拽到列区** | 从手牌区拖拽卡牌到列区成功 | 消耗1步 |
| **列间移动** | 在列区之间移动卡牌 | 消耗1步 |
| **恢复库存** | 牌库抽空后点击"恢复库存"将手牌放回牌库 | 消耗1步 |

#### 2.3.2 列区操作规则

**列区状态定义**:

| 状态 | 说明 |
|------|------|
| **列区-空白** | 列区没有任何卡牌 |
| **列区-分类卡** | 列区最下面一张为分类卡,只能移动到空白的分类槽或万能卡上 |
| **列区-单词卡** | 列区最下面一张为单词卡,可拖拽放置到同词组的单词卡上或分类槽上的分类卡上 |
| **列区-万能卡** | 列区最下面一张为万能卡,允许拖拽到其他列区位置 |

**界面规则**:
- 列区卡牌从上到下排列,y轴位置最下方的卡牌为翻开状态,可见全部信息且可拖拽
- 其余卡牌均为卡背状态,不可查看信息,不可拖拽
- 翻开的卡牌被成功拖拽放置后,下一张卡背状态的卡牌变为翻开状态;若已无卡牌,则该列区变为**空白状态**

**列区初始化规则**:
- 关卡中每个列区可配置默认卡牌数量
- 初始卡牌按照从上到下的顺序放置在列区中

**列区-分类卡拖拽规则**:

**拖拽时的视觉规则**:
- 点击拖拽时,选中该分类卡及同词组的所有卡牌一同拖拽
  - 若一同拖拽的卡牌数量 > 1:卡牌折叠在一起,只显示最下方单词卡的文本信息,右上角显示本次拖拽的卡牌数量
  - 若卡牌数量仅为 1:不显示数字

| 目标位置 | 操作结果 | 判定 |
|---------|---------|------|
| 空白的分类槽 / 空白的列区 / 万能卡上 | ✅ 允许放置,步数-1 | 有效操作 |
| 列区-单词卡(同词组) | ✅ 允许放置,步数-1 | 有效操作 |
| 列区-单词卡(不同词组) | ❌ 无法移动,提示:**分类卡只能放在与其类别相符的词语上** | 错误操作 |
| 列区-分类卡 | ❌ 无法移动,提示:**此列以分类卡开始,你只能将卡槽上将卡牌堆叠在分类卡上** | 错误操作 |
| 分类槽(已有卡牌放置) | ❌ 拖拽无效,自动还原到原位置 | 错误操作 |
| 其他区域 | ❌ 拖拽无效,自动还原到原位置 | 无效操作 |

**列区-单词卡拖拽规则**:

**拖拽时的视觉规则**:
- 点击拖拽时,选中列区最下方的单词卡,并将该列中同词组的所有卡牌一同选中
  - 若选中卡牌数量 > 1:卡牌折叠,只显示最下方单词卡的文本信息,右上角显示本次拖拽的卡牌数量
  - 若卡牌数量仅为 1:不显示数字

| 目标位置 | 操作结果 | 判定 |
|---------|---------|------|
| 分类槽(已放置相同词组的分类卡或单词卡) | ✅ 允许放置,步数-1 | 有效操作 |
| 分类槽(空白) | ❌ 无法放置,提示:**只有分类卡才能放置在分类槽上** | 错误操作 |
| 分类槽(已放置不同词组的卡牌) | ❌ 无法放置,自动还原到原位置 | 错误操作 |
| 列区-空白 | ✅ 允许放置,步数-1 | 有效操作 |
| 列区-分类卡(同词组) | ✅ 允许放置,步数-1 | 有效操作 |
| 列区-分类卡(不同词组) | ❌ 无法移动,提示:**此列以分类卡开始,你只能将卡槽上将卡牌堆叠在分类卡上** | 错误操作 |
| 列区-万能卡 | ✅ 允许放置,步数-1 | 有效操作 |
| 列区-单词卡(同词组) | ✅ 允许放置,步数-1 | 有效操作 |
| 列区-单词卡(不同词组) | ❌ 无法移动,提示:**你只能堆叠同一词组的卡牌** | 错误操作 |

**列区-万能卡拖拽规则**:

- 选中万能卡拖拽时:
  - 若最下方**只有万能卡**:可移动到其他列区
  - 若最下方**还有其他分类卡或单词卡**:❌ 无法移动,提示:**无法使用万能卡来移动牌堆**

| 目标位置 | 操作结果 | 判定 |
|---------|---------|------|
| 列区-空白 | ✅ 允许放置,步数-1 | 有效操作 |
| 列区-分类卡 | ✅ 允许放置,步数-1 | 有效操作 |
| 列区-单词卡 | ✅ 允许放置,步数-1 | 有效操作 |
| 分类槽 | ❌ 无法放置,提示:**万能卡不能放置在分类槽上** | 错误操作 |
| 万能卡上 | ❌ 无法放置,提示:**万能卡不能叠加** | 错误操作 |
| 其他区域 | ❌ 拖拽无效,自动还原到原位置 | 无效操作 |

#### 2.3.3 分类槽操作规则

**界面规则**:

| 状态 | 显示内容 |
|------|---------|
| 空白 | 显示默认分类槽图片 |
| 有分类卡,无单词卡 | 显示分类卡完整信息:左上角为分类卡标识,右上角为`0/N`(N为该词组最大卡牌数量),中间为分类卡文本信息 |
| 有分类卡,有单词卡 | 显示最新放置的单词卡信息;单词卡左上角显示`Y/N`(Y为当前已放置数量,N为该词组最大数量),分类槽上方新增分类卡文本信息 |
| 分类卡对应单词卡数量已满 | 播放卡包收集动画,分类槽变更为空白状态,可以重新放置新的分类卡 |

**操控规则**:
- 分类槽为**空白**时:只能放置分类卡,放置后不允许拖拽移动
- 分类槽有分类卡后:只能放置与分类卡**相同词组**的单词卡
- 当最后一张卡牌被放入分类槽后,该槽位变更为空白状态,判定为该词组完成
- 当所有分类槽都完成时,弹出通关奖励界面

#### 2.3.4 手牌区操作规则

**界面规则**:

| 状态 | 显示方式 |
|------|---------|
| 无手牌 | 显示默认空白区域 |
| 1张手牌 | 完整显示该手牌内容 |
| 2张手牌 | 最新手牌完整显示,前一张手牌隐藏在后面只显示侧边部分,内容竖向显示在侧边 |
| 3张及以上手牌 | 只显示3张:最新手牌完整显示,前2张重叠只显示侧边部分,内容竖向显示在侧边;第3张之前的手牌隐藏 |

**操控规则**:
- 每获得1张手牌,手牌区数量+1,**最多显示3张**,超过3张时将之前的卡牌隐藏
- **只有最新的一张手牌允许拖拽**
- 手牌被成功拖拽至分类槽或列区后,手牌区数量-1,后续手牌自动向前补位;若有隐藏手牌,则按顺序从隐藏手牌中抽出显示
- 拖拽规则与**列区的卡牌操控规则**相同

**手牌拖拽特殊规则**:
- 手牌区每次只拖拽1张(最新手牌),不会出现多张拖拽的情况
- 手牌没有"列区-万能卡"状态,因为手牌区不保存万能卡

#### 2.3.5 牌库操作规则

**界面规则**:
- 关卡内所有词组卡牌中,未放置在列区和手牌区的**剩余卡牌**
- 牌库中的卡牌在关卡开始时由洗牌算法生成固定顺序序列,之后不再改动
- 牌库默认为**卡背状态**
- 牌库右上角显示当前牌库中的卡牌总数量

**操控规则**:
- 点击牌库卡牌:从牌库顺序抽出1张手牌至手牌区,牌库数量-1,步数-1
- 当最后一张卡牌被抽出后,牌库显示**"恢复库存"**文本
  - 点击该文本:将手牌区所有手牌按初始顺序放回牌库并恢复为卡背状态,步数-1

#### 2.3.6 连击计算规则

**有效操作定义**:
- 所有可以成功放置卡牌的操作都属于有效操作
- 包括:翻牌、拖拽到分类槽、拖拽到列区、列间移动、恢复库存等成功执行的步骤

**连击规则**:
- **连击**:每完成1次**有效操作**,连击次数+1
- 期间不允许有**错误操作**,若发生错误操作,则当前累计的连击次数**重置为0**
- **无效操作**(拖拽到非目标区域自动还原)不影响连击次数计算

#### 2.3.7 过关/失败判定

| 判定条件 | 结果 |
|---------|------|
| 所有分类槽都完成(所有词组的卡牌都已正确分类) | ✅ 关卡胜利,弹出通关奖励界面 |
| 步数耗尽且仍有未完成的分类槽 | ❌ 关卡失败 |

**操作流程说明**:
1. 玩家点击牌库翻牌到手牌堆(消耗1步)
2. 将手牌堆或列区的卡牌拖拽到对应分类槽(消耗1步)
3. 可以从手牌堆拖拽卡牌到列区进行整理(消耗1步)
4. 可以在列区之间移动卡牌整理(消耗1步)
5. 使用道具:提示(💡)高亮可匹配卡牌、撤回(↩️)撤销上一步、万能牌(🃏)可匹配任意槽
6. 重复操作直到所有分类槽完成或步数耗尽

**通关流程**:
1. 将分类卡拖拽到空白的分类槽
2. 将同词组的单词卡拖拽到该分类槽
3. 当该词组的所有卡牌都放入分类槽后,播放卡包收集动画,分类槽变更为空白状态
4. 重复步骤1-3,直到所有词组完成
5. 当最后一个词组完成时,弹出通关奖励界面

#### 2.3.8 错误提示汇总

| 场景 | 提示文本 |
|------|----------|
| 分类卡拖拽到不同词组的单词卡上 | **分类卡只能放在与其类别相符的词语上** |
| 单词卡拖拽到不同词组的分类卡上 | **此列以分类卡开始,你只能将卡槽上将卡牌堆叠在分类卡上** |
| 单词卡拖拽到不同词组的单词卡上 | **你只能堆叠同一词组的卡牌** |
| 分类卡拖拽到已有卡牌的分类槽 | ❌ 拖拽无效,自动还原到原位置 |
| 单词卡拖拽到空白分类槽 | **只有分类卡才能放置在分类槽上** |
| 单词卡拖拽到不同词组的分类槽 | ❌ 拖拽无效,自动还原到原位置 |
| 万能卡上有其他卡牌时拖拽 | **无法使用万能卡来移动牌堆** |
| 万能卡拖拽到分类槽 | **万能卡不能放置在分类槽上** |
| 万能卡拖拽到另一张万能卡上 | **万能卡不能叠加** |
| 拖拽到非目标区域 | ❌ 拖拽无效,自动还原到原位置 |

## 三、关卡系统模块

### 3.1 关卡结构

| 关卡阶段 | 卡牌数 | 类别数 | 最大步数 | 布局 | 特点 |
|:-------:|:-----:|:-----:|:-------:|:----:|------|
| 第1-2关（引导） | 25-30 | 5-6 | ∞ 无限 | 3×3 | 新手教学，无失败惩罚 |
| 第3-6关（过渡） | 33-48 | 6-7 | 145-150 | 4×4 | 引入步数限制 |

### 1. 组件查找规范 (component-finder-usage.mdc)

**强制等级**: 🔴 必须遵守  
**适用范围**: 所有 Unity UI 组件和预制体

**核心要点**:
- 必须使用 `ComponentFinder` 工具类进行组件查找
- 禁止使用 `transform.Find().GetComponent()` 原生方式
- 查找操作必须在 `Awake()` 或 `Start()` 中执行
- 默认开启缓存机制，提高性能
